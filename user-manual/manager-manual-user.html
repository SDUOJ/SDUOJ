# 用户管理

# 页面入口

# 修改用户基本信息

点击 Edit，即可弹出一个模态框，用于动态修改用户的基本信息。

# 修改用户密码

点击 Password，即可弹出一个模态框，用于动态修改用户的密码。

# 添加新用户

点击表格左下角的 ADD 按钮，即可弹出一个模态框，用于添加新用户。其中有红点标注的字段为必填字段。

# 删除用户

选中表格第一列的单选框，即可选择性地删除用户。

# 批量导入用户

# 文件导出

选中表格第一列的单选框，即可选择性地导出用户信息。
